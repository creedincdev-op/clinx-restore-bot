Frequently Asked Questions

How do I add CLINX to my server?+

Click any Add To Discord button on this page and authorize CLINX using your server admin account.

What can CLINX include in backups?+

Roles, categories, channels, permission overwrites, and selected server configuration values.

Can I restore without deleting existing channels?+

Yes. Use restore-missing mode or keep destructive options disabled during backup load selection.

Can I paste a huge channel list from ChatGPT or Notepad?+

Yes. Use /masschannels with category headers plus optional channel descriptions or topics.

Where are backups stored?+

Current restore workflows store JSON backup snapshots locally so you can reuse backup IDs and run repeatable loads.
